# Settings

Atualizado em: 2 de abril de 2026
Group: Navegação
Status: Draft

## O que é

Área de configuração da conta do usuário. Cobre perfil pessoal, billing e gerenciamento de usuários da organização.

## Por que existe

Configurações de conta são separadas das configurações de produto (conectores, skills, KB). Settings é o espaço de administração do usuário como entidade — não do fluxo de trabalho.

## Como existe

**Seções:**

**Profile**

- Nome, foto, e-mail, idioma preferido
- Configurações de notificação
- Autenticação (2FA, sessões ativas)

**Billing**

- Plano atual e limites de uso
- Histórico de faturas
- Método de pagamento
- Upgrade/downgrade de plano
- Billing é obrigatório desde o dia 1 — sem exceção (princípio PLG)

**Users**

- Lista de usuários da organização com permissões individuais
- Convite de novos usuários
- Permissões por objeto (referência Figma/Notion — não grupos rígidos)
- Status: ativo, pendente, inativo

**Comportamento:**

- Acessível via Navbar
- Não faz parte do fluxo de trabalho — é configuração de suporte
- Alterações em billing com confirmação explícita antes de aplicar

## Onde é usado

Seção separada acessada pela Navbar.

## Quem usa

- Profile: qualquer usuário
- Billing: admin da conta / owner
- Users: admin da conta / owner
