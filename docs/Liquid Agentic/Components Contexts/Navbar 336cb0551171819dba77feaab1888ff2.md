# Navbar

Atualizado em: 2 de abril de 2026
Group: Navegação
Status: Draft

## O que é

Barra de navegação lateral (sidebar) que dá acesso a todas as seções do produto. É a estrutura de orientação global — o usuário sempre sabe onde está e o que está disponível.

## Por que existe

O produto tem múltiplas dimensões (conversas, agentes, mercado, conectores, skills, contexto) que precisam ser acessíveis sem interromper o fluxo do chat.

## Como existe

**Estrutura da navbar (de cima para baixo):**

- [ + New Task ]
- Recents (Listagem das conversas recentes)
- Agents (Listagem dos Entity Agents ativos)
- Market / Bloomberg
- Conectores / Create new connector
- Knowledge Base / Files + Docs
- Skills / Create new skill
- Account Switch
- Settings (Profile / Billing / Users)

**Comportamento:**

- Colapsável — pode ser minimizada para expandir a área do chat
- Seções expansíveis com subnavegação
- Item ativo destacado visualmente
- Indicadores de status: conectores com erro, skills pendentes de aprovação

## Onde é usado

Presente em todos os layouts. Lateral esquerda.

## Quem usa

Usuário final para navegar entre seções.
