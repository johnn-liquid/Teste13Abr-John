# Text

Atualizado em: 2 de abril de 2026
Group: Chat/Interação
Status: Draft

## O que é

Bloco de texto gerado pelo agente como resposta a uma mensagem. É o output mais simples — prosa, explicações, resumos, confirmações.

## Por que existe

Nem todo output precisa ser um artefato estruturado. Respostas de texto existem para comunicação direta: confirmar uma ação, explicar um resultado, fazer uma pergunta de clarificação.

## Como existe

**Formatos suportados:**

- Markdown (negrito, itálico, listas, links)
- Código inline
- Citações de fontes/conectores utilizados

**Comportamento:**

- Renderizado diretamente no fluxo da conversa (inline no chat)
- Pode conter CTAs inline (botões de ação embutidos no texto)
- Gerado em streaming — aparece progressivamente
- Pode vir acompanhado de artefatos quando o conteúdo exige visualização

**Regras:**

- Texto nunca substitui um artefato quando o output é estruturado
- Linguagem sempre orientada à ação
- Em análises de crédito, o texto é orientado a aprovação, nunca a reprovação

## Onde é usado

Dentro do Chat, no fluxo da conversa. Pode aparecer antes, depois ou sem artefato.

## Quem usa

Gerado pelos agentes (principalmente Skill Agents e Orchestrator). Lido pelo usuário.
