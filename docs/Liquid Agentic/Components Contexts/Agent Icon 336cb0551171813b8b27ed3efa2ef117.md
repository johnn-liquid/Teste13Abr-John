# Agent Icon

Atualizado em: 2 de abril de 2026
Group: Chat/Interação
Status: Draft

## O que é

Representação visual do agente que está respondendo ou atuando em determinado momento. Aparece associado a mensagens, indicadores de processamento e outputs gerados.

## Por que existe

O sistema tem múltiplos agentes operando (Entity, Skill, Orchestrator, Liquidity). O usuário precisa de uma âncora visual para entender a origem de cada resposta — especialmente em fluxos onde mais de um agente participou.

## Como existe

**Estados:**

- `idle` — ícone estático, agente disponível
- `active` / `processing` — indicação animada de que o agente está trabalhando
- `completed` — output entregue

**Variações por tipo de agente:**

- Agente principal (Orchestrator) — identidade visual padrão do produto
- Skill Agent — pode ter variação visual por tipo de skill (análise, documento, pricing)
- Entity Agent — associado à entidade monitorada (ex: ícone de empresa)
- Liquidity Agent — distinção visual por ser o agente estratégico

**Comportamento:**

- Aparece sempre à esquerda das mensagens do agente
- Em fluxos encadeados, o ícone do agente muda conforme quem está respondendo
- Pode ser clicável para revelar qual skill/conector foi utilizado

## Onde é usado

Dentro do Chat, junto a cada mensagem ou bloco de output gerado pelo sistema.

## Quem usa

Exibido para o usuário. Gerado automaticamente pelo sistema com base no agente ativo.
