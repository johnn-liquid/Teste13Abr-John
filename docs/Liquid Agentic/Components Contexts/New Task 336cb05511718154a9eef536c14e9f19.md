# New Task

Atualizado em: 2 de abril de 2026
Group: Navegação
Status: Draft

## O que é

Ação que inicia uma nova conversa/sessão em branco. É o ponto de entrada de cada interação com o sistema.

## Por que existe

Cada tarefa é uma sessão independente — com seu próprio contexto, participantes e histórico. "New Task" separa intenções distintas em threads organizadas.

## Como existe

**Comportamento:**

- Botão sempre visível no topo da navbar
- Ao clicar: abre o chat em estado `idle`, sem histórico, com placeholder contextual no User Input
- A sessão anterior é automaticamente salva em Recents
- Pode receber contexto inicial pré-configurado

**Estados do botão:**

- `default` — disponível
- `active` — uma task sem título está em andamento (feedback visual para evitar duplicatas)

## Onde é usado

Topo da Navbar.

## Quem usa

Usuário final para iniciar novas interações.
