# Artifact — KPI

Atualizado em: 2 de abril de 2026
Group: Artefatos
Status: Draft

## O que é

Componentes de dado que exibem um valor numérico isolado com contexto visual. São o formato mais simples de artefato — uma métrica com significado imediato, sem necessidade de gráfico ou tabela.

## Por que existe

O mercado imobiliário e financeiro opera com indicadores-chave que precisam ser consumidos em segundos: taxa de aprovação, valor total da carteira, inadimplência, score médio. O KPI entrega um número com contexto sem exigir interpretação.

## Como existe

### Variantes

| Variante | Quando usar |
| --- | --- |
| `kpi-simple` | Um número com rótulo. Ex: "R$ 2.4M — Valor total da carteira" |
| `kpi-delta` | Número + variação em relação a período anterior (positivo/negativo). Ex: "87% ↑ +3pp vs. mês anterior" |
| `kpi-sparkline` | Número + mini-gráfico de tendência histórica ao lado. Ex: "12% inadimplência [linha descendente]" |

**Tamanhos:**

- `sm` — usado em grupos de KPIs (dashboard) ou dentro de cards
- `md` — destaque individual em painel split
- `lg` — KPI hero, destaque máximo em dashboard dominante

**Estados:**

- `loading` — skeleton enquanto dado é carregado
- `stale` — dado fora do período esperado de atualização (indicador visual sutil)
- `error` — falha no conector, dado indisponível

### Tokens usados

**Cor:**

- Valor principal: `color-text-primary`
- Rótulo: `color-text-secondary`
- Delta positivo: `color-feedback-success`
- Delta negativo: `color-feedback-error`
- Neutro/sem variação: `color-text-tertiary`

**Tipografia:**

- Valor (sm): `type-heading-md` / bold
- Valor (md): `type-heading-lg` / bold
- Valor (lg): `type-heading-xl` / bold
- Rótulo: `type-label-sm` / regular
- Delta: `type-label-sm` / medium

**Espaçamento:**

- Padding interno: `space-12` (sm), `space-16` (md/lg)
- Gap rótulo → valor: `space-4`
- Gap valor → delta: `space-8`

### Regra de quando usar

- Use `kpi-simple` quando o número fala por si só e não há comparação relevante
- Use `kpi-delta` quando a variação temporal é o dado mais importante (performance, tendência)
- Use `kpi-sparkline` quando o usuário precisa ver o padrão histórico sem abrir um gráfico completo
- **Nunca use KPI para dados que precisam de contexto qualitativo** — nesses casos, use `artifact-card`
- Em grupos, use no máximo 4 KPIs por linha para não fragmentar o foco

## Onde é usado

- Painel de mercado (Market): grupos de KPIs por região ou segmento
- Artifact — Card: como atributo numérico dentro de um card
- Dashboard dominante: KPIs hero acima de gráficos

## Quem usa

Gerado pelos Entity Agents (dados de mercado em tempo real) e Skill Agents (métricas de análise). Consumido por analistas e gestores para monitoramento rápido.
