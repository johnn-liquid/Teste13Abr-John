Group: Artefatos
Status: Draft

## O que é

Artefato de formato longo — relatórios, laudos, análises detalhadas, pareceres. É o output mais completo que uma skill pode gerar.

## Por que existe

Decisões financeiras complexas precisam de documentação auditável. O document artifact é o registro formal do trabalho da skill — pode ser impresso, assinado, arquivado ou enviado para terceiros.

## Como existe

**Estrutura:**

- Título + metadados (data, skill utilizada, versão, fontes)
- Seções com hierarquia (H1/H2/H3)
- Texto, tabelas, gráficos inline
- Assinatura digital / referência de auditoria no rodapé

**Tipos comuns:**

- Relatório de análise de crédito
- Laudo de viabilidade de empreendimento
- Relatório de due diligence
- Parecer de underwriting

**Comportamento:**

- Gerado no painel dominante ou split
- Exportável como PDF
- Compartilhável via link com permissão controlada
- Versionado — nova análise gera nova versão, não substitui a anterior
- Referenciável em outras conversas

## Onde é usado

Layout split 50/50 ou dashboard dominante. Raramente inline no chat.

## Quem usa

Gerado pelos Skill Agents. Consumido pelo analista, corretor, e em processos de auditoria.
