# Voice

Atualizado em: 2 de abril de 2026
Group: Chat/Interação
Status: Draft

## O que é

Modalidade de entrada e/ou saída por voz. Permite ao usuário interagir com o agente falando, e ao agente responder em áudio.

## Por que existe

Amplia o canal de entrada além do texto. Útil em contextos onde digitar é impraticável (mobile, campo, reunião) ou onde a fluidez de uma conversa falada agrega valor.

## Como existe

**Modos:**

- `input` — usuário fala, sistema transcreve e processa como User Input de texto
- `output` — agente responde em voz sintetizada (TTS)
- `bidirecional` — conversa contínua em tempo real

**Estados:**

- `idle` — microfone disponível, não ativo
- `listening` — gravando input do usuário
- `processing` — transcrevendo / processando
- `speaking` — agente respondendo em áudio

**Comportamento:**

- Transcrição exibida no chat como User Input normal
- Output de voz pode ocorrer em paralelo com a exibição de texto
- Usuário pode interromper o agente enquanto ele fala

## Onde é usado

Integrado ao User Input (botão de microfone). Pode ser ativado como modo de sessão.

## Quem usa

Usuário final, especialmente em mobile ou em contextos hands-free.
