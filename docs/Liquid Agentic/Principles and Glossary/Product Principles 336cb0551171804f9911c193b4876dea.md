# Product Principles

Created: 2 de abril de 2026 11:24

| Princípio | Impacto |
| --- | --- |
| **Global** | Multi-idioma e multi-moeda desde o dia 1. Começa em inglês. |
| **Product-Led Growth** | Produto vende sozinho. Jornada autônoma, login social, billing nativo. |
| **Agent First** | Arquitetura baseada em agentes. Toda experiência passa pelo orquestrador. |
| **Liquidity Driven** | Toda ação é pipeline para liquidez. Receita principal vem de taxas transacionais. |
| **Regulated Market** | Responsabilidade sobre dados financeiros. Rastreabilidade obrigatória. |
| **Multiplayer** | Uso simultâneo multi-usuário com permissões por escopo. |
